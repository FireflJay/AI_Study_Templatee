AI+ 结构化学习平台

主要功能：

一、AI聊天系统：
1.支持通义千问、DeepSeek等多模型对话；
2.上下文记忆与多轮对话能力；
3.代码/文本多模态输入支持。

二、模块化学习体系：

├── 基础认知：AI概念与伦理
├── 实战指南：PyTorch/TensorFlow框架
├── 办公自动化：文档智能处理
├── 创作增强：
│   ├─ 文本生成
│   ├─ 图像编辑
│   └─ 视频处理
└── 高阶应用：
    ├─ 编程辅助
    └─ 自定义AI Agent开发
    
三、工具矩阵推荐：
按场景分类推荐工具：
文本 | 图像 | 视频 | 数字人 | 编程
示例： ONNX模型转换工具、Stable Diffusion图像生成等；

四、智能客服系统：
混合知识库支持：
1.本地知识库+联网搜索；
2.分步骤问题引导；
3.个性化学习路径推荐。
